FrozenDelegate(initialValue: T)

A delegate to create thread safe frozen properties backed by AtomicReference. The property can be declared with by [FrozenDelegate] (initialValue) keyword. All get and set operations are then performed atomically. Note that the values are frozen and become immutable once set. So any updates will cause a failure.
